# AET313_SP26_P2
Spring 2026 AET 313 Project 2

Technical/Interactive Lead - Heather Kim
Art/Narrative Lead - Rhiannon Ferguson

🎭 Romeo & Juliet: Interactive Dollhouse
This project is an interactive puppet house that combines Arduino and Teensy to bring the classic tragedy of Romeo and Juliet to life through motion, light, and sound.

🛠 Hardware & Components
Arduino Uno: Controls 4 servo motors for mechanical movements and manages the WS2812B (NeoPixel) LED strips for atmospheric lighting.

Teensy 4.0 + Audio Shield: Handles high-quality audio playback of WAV files, including background music (BGM) and sound effects (SFX).

Copper Tape (Capacitive Touch): Used to create custom touch-sensitive "prop" buttons for user interaction.
